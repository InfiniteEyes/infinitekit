Kali Linux Toolkit for Red/Blue Team Operations

Access popular Kali Linux tools for security assessment and defense.

These commands simulate popular Kali Linux tools. They provide educational information about each tool's purpose and typical usage.
Red Team Tools
Nmap
Network mapping

Network scanning and host discovery tool.

Exploitation

Advanced open-source platform for developing, testing, and executing exploits.

msfconsole
Burp Suite
Web assessment

Web vulnerability scanner and intercepting proxy.

burpsuite
Aircrack-ng
Wireless audit

Suite for wireless network security assessment.

aircrack-ng [options] <capture file(s)>
SQLmap
SQL injection

Automated SQL injection and database takeover tool.

sqlmap -u "http://example.com/page.php?id=1" --dbs

Security auditing and hardening tool for Unix/Linux systems.
